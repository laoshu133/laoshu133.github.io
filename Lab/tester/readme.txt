/---------------快捷键----------------/
Ctrl + Enter -  运行代码，同时按住Alt可以新窗口运行
Alt + Q - 编辑器光标处快速插入JQ1.3.2
Alt + S - 编辑器光标处快速插入<script />标签
Alt + C - 编辑器光标处快速插入<style />标签
Alt + R - 重置编辑区布局
Alt + N - 新建空白文档
Alt + F - 调用编辑器查找
